# Our Pets

ご自身のペットを思う存分に自慢していただく投稿サイトです。
動物と触れ合えるおすすめの施設をユーザーさんに登録していただくことでおすすめ施設を共有できます。

#　URl

# 使用技術

・Ruby
・Ruby on Rails
・Rspec
・Google Maps API
・
・

# 機能一覧

・ユーザー登録、ログイン機能(devise)
・投稿機能
  - 画像投稿(Activestorage)
  - 施設の所在地表示(geocoder)
・画像スライド(bootstrap)
・いいね機能(Ajax)
・ページネーション機能(kaminari)
